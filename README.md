Campus Info Hub - But Teodora
1. Ce este o resursă (resource) în aplicația ta?
O resursă este un obiect sau o entitate accesibilă printr-un URL, care conține informații despre campus, cum ar fi Biblioteca, Cantina sau Evenimentele.

2. Exemplu de URI și explicația componentelor
Exemplu: https://numeutilizator.github.io/campus-info-hub/pages/library.html#schedule

https → protocol securizat
numeutilizator.github.io → domeniul
/campus-info-hub/pages/library.html → calea către pagina Bibliotecii
#schedule → fragment link, duce direct la secțiunea „program”
3. Care părți sunt statice și care sunt dinamice?
Statice: HTML, CSS, imagini
Dynamice: lista de resurse încărcată cu JavaScript din resources.json, filtrarea resurselor, afișarea programului
