Ce este o resursă (resource) în aplicația la?
O resursă este orice element accesibil prin web, cum ar fi o pagină, o imagine sau date (ex: profil utilizator sau produs).

Dă exemplu de un URI și explică componentele acestuia
Exemplu: https://exemplu.ro/produse?id=10

https – protocolul

exemplu.ro – domeniul

/produse – calea către resursă

?id=10 – parametri (identifică resursa exactă)

Care părți sunt statice și care sunt dinamice?

Statice: HTML, CSS, imagini (nu se schimbă des)

Dinamice: date din baze de date, conturi utilizatori, rezultate căutare

Este aplicația ta document-centric sau interactivă (sau ambele)? De ce?
Este ambele, deoarece are pagini informative (document-centric) și funcții interactive precum formulare sau conturi utilizatori.
